FAQs

What is Frontend Mentor,and how will it help me ?

Frontend Mentor offers realistic coding challenges to help developers improve their frontend coding skills with projects in HTML,CSS, and JavaScript. It's suitable for all levels and ideal for portfolio building

Is Frontend Mentor free?

The majority of Frontend Mentor challenges are free, yes. Frontend Mentor do have some that are premium and require a Pro subscription to access. It will say on each challenge whether they are free or premium, so it's easy to tell the difference.

Can I use Frontend Mentor project in my portfolio ?

Definitely! Please do feel free to use whatever you build in your portfolio. Helping developers add professional-looking projects to their portfolio was one of the reasons Frontend Mentor created this platform!

How can I get help if i'm stuck on a challenge?

The best (and quickest) way to get help on a challenge is in our Discord server. There are thousands of other developers in there, so it's a great place to ask questions. Frontend Mentor even have a dedicated "help" channel! If you haven't joined yet, you can get an invite to our Discord server here.
